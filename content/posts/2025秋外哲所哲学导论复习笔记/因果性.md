# 因果性（Causation）复习笔记

> 本笔记系统总结当代因果哲学中的主要理论、核心论证与关键反例，重点对比还原论与反还原论路径，并标出各理论在抢先、过度决定、负面因果等难题上的表现。

---

## 一、休谟传统：规律性理论（Regularity Theory）

**核心思想**
- 因果性 = 时间先后 + 空间邻近 + 恒常联结（规律性相继）
- 排除“必然联系”的形而上学承诺

**主要问题**
1. **过于宽松**：琐屑规律（如“按开关 → 月亮未爆炸”）
2. **共同原因**：相关≠因果 (冰激凌销量高，中暑人数增加)
3. **方向性缺失**：无法解释为何是 A 导致 B
4. **事件类型非中立**：规律依赖描述粒度与理论背景

---

## 二、Mackie：INUS 条件理论

**INUS 定义**
- 原因是“不足但不可或缺的部分”，属于“非必要但充分的条件组”

**改进**
- 贴合多因素因果
- 通过“不可或缺性”削弱琐屑规律
- 容纳多重实现与过度决定

**遗留问题**
- **过于宽松**：条件组可任意拼装
- **背景/原因区分**需诉诸“因果场（causal field）”
- **方向性与共同原因**仍不稳
- 本体论上仍接近规律主义

---

## 三、反事实因果论（Lewis）

### 3.1 基本语义
- 反事实：`p □→ q` 为真  
  iff 在**相对于现实世界**的**最邻近 p 世界**中，q 为真
- “最邻近”基于整体相似性（保自然定律、最小改动）

### 3.2 版本演进
- **版本1**：因果 = 直接反事实依赖  
  → 失败于**抢先（preemption）**：约定先后扔石头
- **版本2**：因果 = **实际发生的反事实依赖链**  
  → 解决**早抢先*:约定先后扔石头*
- **版本3（影响）**：因果 = **对结果实现方式的反事实影响**  
  → 解决**晚抢先**:一起扔石头，有快有慢

### 3.3 典型难题
- **抢先**：需区分实际路径
- **过度决定**：完全对称冗余下，差异消失  
  → Lewis：相关直觉不可靠
- **负面因果**：可处理，但依赖“最近世界”选择

---

## 四、概率因果论（Reichenbach）

**核心公式**
- `C` 是 `E` 的原因 iff `P(E|C) > P(E|¬C)`

**问题**
1. **参考类问题**：概率依赖所选背景
2. **抑制/预防**：原因未必提高结果概率（如避孕）
3. 概率抬高既非必要也非充分

**Lewis 的回应**
- 概率是因果在**可靠系统**中的表征  
- 因果**先于**概率，而非由概率定义

---

## 五、过程论：Salmon → Dowe

### 5.1 Salmon：标记转移
- 因果过程能接收并传递“标记”
- 区分真实过程 vs 伪过程（影子等）

**问题**
- “标记”概念模糊、可人为注入

### 5.2 Dowe：守恒量转移
- 因果 = **守恒量**（能量、动量等）的转移
- 明确、物理化、客观

**优势**
- 生成性（productive causation）
- 物理抢先/落空易判

**代价**
- **负面因果**（撤除阻碍、未能成行）被否认或降格
- 适用性局限于**物理因果**

---

## 六、负面因果（Negative Causation）

**类型**
- 撤除阻碍（removal of preventer）
- 未能成行（omission）
- 无过程交叉的因果判断

**分歧**
- 过程论：否认其为“真正因果”
- 反事实/干预论：以**差异制造/可控性**吸收
- **分层立场**（推荐）：  
  - 物理层面：过程/转移  
  - 解释与实践层面：差异/控制

---

## 七、因果初始主义（Michael Tooley）

### 一、基本立场：因果性是本体论原语
- **核心主张**：因果关系是世界中的一种**基本事实（primitive fact）**，不能被还原为任何非因果事实。
- 否认因果事实对以下非因果事实的**超随附（supervenience）**：
  - 事件的内在性质
  - 事件之间的时空与关系性质
  - 自然法则（尤其是非决定论法则）
  - 概率分布
  - 反事实真值结构
  - 物理过程或守恒量转移
- “原语”并非神秘化因果，而是主张：  
  **“A 导致 B”不能被定义为或等同于任何更基础的非因果描述。**

---

### 二、论证一：非决定论法则下的因果指派不可还原

#### 1. 设定
- 存在非决定论法则：
  - 若对象 a 具有性质 **P**，则可能导致 **Q 或 R**
  - 若对象 a 具有性质 **S**，则也可能导致 **Q 或 R**
- 法则仅给出可能性或概率结构，并不指定具体因果指向。

#### 2. 实际发生情形
- 初始时刻：a 同时具有 **P 和 S**
- 后继时刻：a 同时具有 **Q 和 R**

#### 3. 核心问题
- **Q 是由 P 导致，还是由 S 导致？**
- **R 又是由谁导致？**

#### 4. 还原性资源的失效
- **法则**：P、S 都允许 Q、R → 无法区分
- **概率**：P、S 都提高 Q、R 的概率 → 无法指派实际因果
- **反事实**：去掉 P 或 S，结果仍可能发生 → 反事实依赖失败
- **过程/机制**：无非任意方式将 Q 与 P 或 S 唯一配对

#### 5. 结论
- 即便所有非因果事实完全给定，**因果指向仍未被决定**。
- 若现实中确有如下事实之一：
  - P → Q 且 S → R  
  - 或 P → R 且 S → Q  
- 则该差异只能由**额外的因果事实**来确定。

---

### 三、论证二：严格复刻情形的可能性（反超随附论证）

#### 1. 论证策略
- 若存在两个可能情形：
  - 在**所有非因果事实上完全一致**
  - 却在**因果事实上不同**
- 则因果性不可还原为非因果事实。

#### 2. 严格复刻设想
- 两个事件 **M** 与 **M\***：
  - 内在性质完全相同
  - 关系性质完全相同（时空位置、法则嵌入、概率位置、反事实结构等）
- 在一切非因果描述下不可区分。

#### 3. 因果差异
- M 是由事件 **P** 导致的
- M\* 并非由 P 导致（可能由 Q 导致，或在非决定论背景下无特定前因地产生）

#### 4. 推论
- 非因果事实完全一致，但因果事实不同  
  → **因果事实不超随附于非因果事实**
- 否定所有试图用法则、概率、反事实或过程来穷尽刻画因果性的还原论。

---

### 四、Tooley 立场的收获
- 在非决定论世界中，**因果指向本身是一个额外的、不可还原的事实维度**。
- 即便完全掌握世界的非因果结构，仍可能不知道**究竟是哪一个事件导致了哪一个事件**。

---

### 五、代价与压力

#### 1. 形而上学代价
- 必须承认：
  - 世界中存在不可分析的**原初因果事实**
- 理论不够节约，本体论负担较重。

#### 2. 认识论难题
- 若因果性是原语：
  - **我们如何认识与验证这些因果事实？**
- 不能依赖概率、反事实或物理过程。
- 通常诉诸：
  - 科学理论的整体解释力
  - 因果法则的系统化
  - 实践中因果判断的稳定性  
- 但认识论路径不如还原论清晰。

## 八、干预论（James Woodward）

**核心判准**
- `C` 是 `E` 的原因 iff  
  存在**理想干预**改变 `C` 的取值，且 `E` 随之改变

**理想干预条件**
- 仅经由 `C` 影响 `E`
- 切断 `C` 的其他原因
- 不破坏 `C→E` 机制

**优势**
- 自然处理**负面因果**
- 与实验、控制、因果推断实践契合
- 系统屏蔽抢先中的备用路径

**与 Lewis V3 的对比**
- Lewis：相似性排序 + 反事实影响（偏形而上）
- Woodward：干预可达性 + 不变性（偏方法论）

**难点**
- 过度决定转化为**变量分解/可干预性**问题

## 九、综合对照（要点表）

| 理论 | 强项 | 代价 |
|---|---|---|
| 规律性 | 简洁 | 琐屑、无方向 |
| INUS | 多因素 | 过宽、语境依赖 |
| 反事实（Lewis） | 差异、抢先 | 过度决定 |
| 概率 | 不确定性 | 参考类 |
| 过程（Salmon/Dowe） | 生成性 | 负面因果 |
| 初始主义（Tooley） | 反还原 | 形而上学成本 |
| 干预论（Woodward） | 可控性、实践 | 变量建模 |
